Демо-проект на Android Studio Panda2:
- для оформления пользовательского интерфейса применен Jetpack Compose
- данные сохраняются в SQLite
- использован паттерн MVVM (ViewModel)
- для внедрения зависимостей иснользован Koin

Функции:
- добавление запись по кнопке +
- просмотр запись по короткому клику (далее - ссылка на редактирование)
- изменение статуса по долгому клику (отображается изменением цвета)
- удаление записи по свйпу вправо/влево
- для установки даты/времени использованы соответствующие диалоговые окна
